Motion spec (all CSS-only)

Durations: 120ms (focus/color) · 150–180ms (hover/press) · 200–240ms (entrances/crossfades) · 300ms max (one-off pops: mint node, toggle chip).
Easing: ease-out for entrances, ease-in-out for state swaps; the two "spring" pops use cubic-bezier(.34,1.56,.64,1).
Catalog entrance: cards fade + rise 8px, 40ms stagger, first 12 only.
Card hover: −3px translate + card shadow; portrait scale 1.03 (transform only, no layout).
Buttons: primary −1,−1 shift + shadow 2→4px; press back to 0.
Filters: dropdown 140ms fade + 6px drop; grid crossfade 180–200ms on tag change.
Forms: focus ring 120ms; error shake ±3px ×2 (200ms) + message slides down 4px.
Toasts: up 16px + fade 240ms in; auto-dismiss 5s with mint progress hairline; errors persist.
Reduced motion: all transforms/staggers off; opacity-only fallbacks; shake replaced by ring flash.
